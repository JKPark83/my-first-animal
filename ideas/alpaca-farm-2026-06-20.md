# 💡 알파카 농장 (Alpaca Farm) — 로블록스 케어/경영 게임

## 한 줄 요약
빈 농장에 떨어진 플레이어가 300달러로 시작해 **10종 알파카(50$~300만$)** 를 사서 울타리 농장에 데려와,
**먹이주기·빗어주기·목욕시키기** 케어와 **양털 패시브 수익**으로 돈을 모아 더 비싼 알파카를 모으고
**10종 도감 완성**을 목표로 하는 멀티플레이 케어/타이쿤 게임.

## 문제 / 타겟 사용자
- **타겟**: 타이쿤·펫 키우기 장르를 좋아하는 로블록스 플레이어(주로 8~14세).
- **채워주는 욕구**: 짧은 케어 루프의 즉각적 보상 + 수집(도감)·성장(더 예쁜/비싼 알파카)이라는 장기 동기.

## 목표 & 성공 지표
- **플레이어 목표(엔드게임)**: 10종 도감 완성 + 농장을 상위 알파카로 가득 채우기.
- **성공 지표**: 평균 세션 시간, 재방문율(D1 리텐션), 도감 진행률, 첫 알파카 구매까지 걸린 시간.

## MVP 범위
**포함**
- 3D 로비/농장: 입장 시 빈 농장에 스폰, 개인 농장 구역(울타리), 알파카 가게 건물 — **멀티(한 서버 다수, 각자 구역)**.
- 코드로 파츠 조립한 **블록형 귀여운 알파카 10종**(등급별 색/크기/장식 차등).
- **케어 시스템**: 알파카 근접 시 화면 중앙 하단 버튼 3개(먹이/빗질/목욕) 활성화 → 케어 시 등급별 보상 + 욕구 리셋.
- **욕구 시스템**: 배고픔/털엉킴/더러움이 ~5분에 걸쳐 천천히 상승, 일정 수준 이상일 때만 케어 가능(욕구 속도 = 자연 쿨다운). 방치 시 양털 수익·행복도 하락.
- **양털 패시브 수익**: 알파카가 60초마다 등급별 양털 생산 → 자동/수집 판매.
- **상점·경제**: 보유 금액 한도 내 1마리씩 구매, 슬롯 제한(시작 3칸, 확장 구매), `GameConfig` 단일 밸런스 파일.
- **DataStore 영구 저장**: 돈·보유 알파카·알파카 상태·도감·슬롯.
- **알파카 이동/행동 AI**: 농장 안을 어슬렁 배회, 상태가 외형(엉킨 털·얼룩)으로 표시.
- **글로벌 리더보드(OrderedDataStore) + 도감 UI**.
- **로벅스 프리미엄**: GamePass/Developer Product 구조 정의(한정 알파카 / 수익 2배 부스트) — 게임 퍼블리시 전까지 토글 가능하게.

**제외(후속)**
- 플레이어 간 알파카 거래/교배, 농장 꾸미기(가구/데코), 미니게임, 시즌/이벤트, 모바일 정밀 최적화.

## 핵심 사용자 플로우
1. 입장 → 빈 개인 농장 구역에 스폰, 300$ 지급, 튜토리얼 힌트.
2. 알파카 가게로 이동 → 10종 목록·가격·능력치 확인 → 보유금 한도 내 구매(처음엔 들풀 알파카 50$).
3. 산 알파카가 내 농장 슬롯에 스폰 → 농장 안 배회 시작.
4. 시간 경과 → 배고픔/털엉킴/더러움 상승(외형 변화) + 양털 누적.
5. 알파카 근처로 가면 하단 케어 버튼 활성화 → 먹이/빗질/목욕 → 욕구 리셋 + 등급별 현금 보상.
6. 양털 수집/판매로 패시브 수입 → 돈 모아 슬롯 확장 & 상위 알파카 구매.
7. 도감 채우기·리더보드 경쟁 반복 → 전설의 알파카 + 10종 도감 완성.

## 기술 접근
- **`my-first-cargame` 구조 그대로 재활용**: Rojo 프로젝트, Luau, `GameConfig`+`Remotes` 패턴, DataStore·OrderedDataStore, 코드 절차적 빌드.
- **ReplicatedStorage**: `GameConfig.luau`(밸런스 전부), `Remotes.luau`.
- **ServerScriptService**: `Main.server`, `DataManager`, `FarmBuilder`, `AlpacaSpawner`, `FarmService`(무리 관리 글루),
  `AlpacaShopService`, `CareService`, `WoolService`, `CollectionService`(도감), `AlpacaAI`, `LeaderboardService`, `GamePassService`.
- **StarterPlayer**: `UIController.client`(상점·HUD·도감·리더보드), `CareController.client`(근접+하단 케어 버튼+양털 수집),
  `AlpacaHud.client`(알파카 머리 위 욕구 게이지).

- 시작 돈 300, 시작 슬롯 3칸(최대 12), 슬롯 확장 비용 점증.
- 욕구는 ~5분에 0→100, 케어 임계치 20(≈60초 자연 쿨다운). 행복도 = 100 - (배고픔+털엉킴+더러움)/3, 양털 수익 0.5~1.0배.
- 성장 페이스: **중간**(하위 티어는 한 세션, 전설은 며칠).

## 첫 구현 단계 (5개)
1. Rojo 스캐폴드 + `GameConfig`(알파카 10종·경제·욕구/슬롯) + `Remotes`.
2. `FarmBuilder`(로비·개인 농장·울타리·가게) + `AlpacaSpawner`(블록형 알파카 등급 외형) + `FarmService`(무리 관리).
3. `AlpacaShopService` + 상점 UI(구매/슬롯 확장).
4. `CareService`(욕구·케어·쿨다운·보상) + `CareController`/하단 버튼 + `AlpacaAI` 배회 + `WoolService` 패시브 + `AlpacaHud` 게이지.
5. `DataManager`(DataStore) + `CollectionService`(도감) + `LeaderboardService` + `GamePassService` 골격 + README.

## 리스크 & 오픈 질문
1. **경제 밸런스**: 케어 vs 양털 vs 가격 곡선이 "중간 페이스"로 맞는지 → 모든 수치를 `GameConfig` 한 곳에서 튜닝.
2. **블록형 알파카 비주얼 품질**: 외부 에셋 없이 파츠만으로 귀엽게 → 등급별 색/장식/크기 + 간단 애니메이션으로 보완.
3. **로벅스 프리미엄**: GamePass/DevProduct는 퍼블리시 후 동작 → MVP는 구조만 + Studio 토글.
